在Termux上安装Hexo博客系统

发表于 2018-10-10 | 更新于 2018-10-12 | 分类于 手机上编程

首先要先安装好Nodejs，然后使用npm安装Hexo，
npm install hexo-cli -g
安装好之后创建博客的目录
hexo init blog
进入刚才创建的目录安装依赖
npm install
启动服务，看一下效果
hexo s
