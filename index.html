通过电脑ssh登录手机termux

发表于 2018-09-30 | 更新于 2018-10-15 | 分类于 手机上编程

假设我们使用的操作系统是Win10，因为Win10自带有ssh相关工具，所以省的我们自己再手动去安装客户端。

一、首先电脑端操作：

  1. 进入CMD命令行输入ssh-keygen -t rsa，生成公钥和私钥文件，碰到要求输入的时候直接回车就行。
  2. 上一步执行完成之后来到你电脑的系统盘，进入“用户”目录，然后进入你名字对应的目录，应该能看到一个.ssh的文件夹，打开后里面有两个文件，一个是id_rsa.pub这个是你的公钥，一个是id_rsa这个是你的私钥。
  3. 把id_rsa.pub公钥文件传输到你的手机download文件夹里。

二、然后我们转到手机上继续操作：

  1. 在termux上安装openssh，输入pkg install openssh。
  2. 为了Termux能读出我们手机上的download文件夹，需要执行termux-setup-storage命令。现在我们的download文件夹就被映射到了~/storage/downloads。
  3. 然后我们通过cat命令把download文件夹里面的公钥文件内容写到authorized_keys文件里，执行如下命令，cat ~/storage/downloads/id\_rsa.pub >> ~/.ssh/authorized_keys。
  4. 上面三部操作完之后现在可以启动手机上ssh服务了，执行sshd命令。
  5. 接下来查看手机的ip地址，输入ifconfig -a，可以看到你手机的ip地址形如192.168.1.2，记下来。
  6. 再查看Termux用户名，输入whoami，能看到形如u0_a82，就是你的用户名，记下来。

三、继续回到电脑上操作：

  1. 输入ssh u0_a82@192.168.1.2 -p 8022，然后一路回车，即可登录到你手机的Termux了。

掌叔的第一部诺基亚手机NOKIA3510

发表于 2018-09-22 | 更新于 2018-10-15 | 分类于 杂谈

大家一看到这个题目肯定一眼就看出这一定是掌叔遥想当年，怀旧式的一篇文章。的确，自久智网新版正式上线以来掌叔终于有一个可以静下心来写点东西的地方。除了分享掌叔开发方面的一些经验以及各种小众系统资讯的基础上，也希望能抒发一下深藏已久的感情。 写成文章一方面是满足掌叔多年以来的一种情怀，另一方面也希望能通过这种方式唤醒屏幕后的你同样心里埋藏的那份情怀。

说回正题，掌叔是在2003年的时候获得了人生中第一部诺基亚手机（掌叔第一步手机是摩托罗拉的v8088），那时候掌叔在上大二，手机是我一位亲戚送的。刚拿到它的时候，觉得样子很搞笑，尤其是外壳那蓝白条纹，这个造型曾被我同宿舍的一位同学戏称为裤衩子手机。你别说看着也确实有点像，再看它那不规则的按键造型，这和当时市面上其它手机相比显得很个性。刚入手的时候除了外观还有一点另我印象特别深刻，就是它的铃声，四和弦，那时候周围人用的手机铃声还都是只能发出滴滴答答的单音，而它四和弦的铃声太悦耳、太美妙了，尤其印象深的是有一首铃声听起来格外的空灵，仿佛身处广袤的宇宙深处。

手机内置的游戏也是一绝，印象最深的两个游戏一个是赛车，还有一个是空间大战。虽然是黑白单色屏幕，不过这俩游戏却做出了当年8位机的水准！有前景有背景，背景还是带卷轴的，想当初掌叔可是玩的不亦乐乎啊！一有空就来上一把。

至于其他功能我印象中是可以通过GPRS上网的，不过通过那个小屏幕上网的意义也不是很大，再加上那时候的网速慢、费用高，所以它的网络功能几乎是没用的。不过当时可以通过发送短信到服务商下载手机铃声，一条铃声一块钱，当时花了几块钱下了几首周杰伦歌做铃声，用了很长时间。好像一直到手机电池彻底充不进电了都没有再换过。 那时候的手机不像现在的智能机，数不清的软件，让人眼花缭乱，那时游戏机就是游戏机、相机就是相机、而手机真的就只是手机。在看看现在，在某种程度上手机已经是电脑+相机+游戏机了。虽然硬件强大、功能丰富，但却再也找不回当初那份单纯的快乐。

你还记得你离上次折腾手机已经过去多久了么？

搭载KaiOS的手机JioPhone2手机已经发布

发表于 2018-09-18 | 更新于 2018-10-15 | 分类于 小众资讯

KaiOS系统的官方网址是：www.kaiostech.com/

JioPhone2是一款类似NOKIA E5那样的直板全键盘手机，它采用了一块240*320像素2.4寸的屏幕，配备了Qwerty全键盘，搭载了一枚1.1Ghz双核处理器，存储为512M内存和4G存储，还支持128GTF卡扩展，配备了前置30万像素后置200万像素镜头，电池容量为2000毫安时，支持4G网络。单从硬件配置来看，像极了当初的NOKIA手机。

可能很多朋友看到这些和今日主流的硬件相比已经十分孱弱的配置后，估计兴趣已然全无了，但是对掌叔来说这么一款低配手机却有着极大的吸引力。首先就是它搭载的KaiOS这个小众系统，大家都知道掌叔是一名不折不扣的小众系统迷，再者掌叔对全键盘更是天生没有抵抗力。其实它和一般意义上的功能机还是有很大不同的，毕竟KaiOS的前身可是FireFoxOS，也是一款智能操作系统呢，可玩性还是很高的。可惜这款手机只在印度发售，如果能在中国上市，掌叔肯定是买买买~
