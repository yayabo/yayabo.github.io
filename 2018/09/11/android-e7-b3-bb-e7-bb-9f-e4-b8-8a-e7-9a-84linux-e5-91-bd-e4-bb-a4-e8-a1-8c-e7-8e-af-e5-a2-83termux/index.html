Android系统上的Linux命令行环境Termux

Termux是一款能够在Android系统上运行的小型Linux环境。只需要安装一个APK文件就能让你的手机立马成为Linux学习利器。 用过Linux的同学都知道，要用好Linux，命令行知识是少不了要学的。包括软件安装、系统更新等等操作都需要通过一行一行的命令来完成。 那么今天就给大家大致演示一下通过Termux来安装Nodejs和PHP。 1、首先下载安装Termux（点击这里下载），启动之后会有一个几十秒的初始化过程，初始化之后就会出现如下界面： 2、输入apt update回车，更新应用源。更新之后一般会提示有哪些软件可以更新了。这时候需要执行pkg upgrade来更新一下软件。 接下来我们来安装nodejs和php。 输入如下命令：（注意每一行命令敲完都要回车）

pkg install nodejs
pkg install php

系统会自动帮你把这个两个软件安装好，最后执行如下两个命令来看一下安装好的Nodejs和PHP的版本。

node -v
php -v

可以看到nodejs的版本是8.11.4，php的版本是7.2.9都是相当新的版本，其实termux里面的软件版本更新是非常频繁的，大家可以时不时的apt update一下看看。 最后我们现在就可以愉快的在手机上学习Linux啦~
