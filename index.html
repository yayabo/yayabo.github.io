通过电脑ssh登录手机termux

假设我们使用的是Windows10操作系统，因为Win10自带ssh工具，省的我们再去安装客户端。 一、首先电脑端操作：

  1. 进入CMD命令行输入ssh-keygen -t rsa，生成公钥和私钥文件，碰到要求输入的时候直接回车就行。
  2. 上一步执行完成之后来到你电脑的系统盘，进入“用户”目录，然后进入你名字对应的目录，应该能看到一个.ssh的文件夹，打开后里面有两个文件，一个是id_rsa.pub这个是你的公钥，一个是id_rsa这个是你的私钥。
  3. 把id_rsa.pub公钥文件传输到你的手机download文件夹里。

二、然后我们转到手机上继续操作：

  1. 在termux上安装openssh，输入pkg install openssh。
  2. 为了Termux能读出我们手机上的download文件夹，需要执行termux-setup-storage命令。现在我们的download文件夹就被映射到了~/storage/downloads。
  3. 然后我们通过cat命令把download文件夹里面的公钥文件内容写到authorized_keys文件里，执行如下命令，cat ~/storage/downloads/id_rsa.pub >> ~/.ssh/authorized_keys。
  4. 上面三部操作完之后现在可以启动手机上ssh服务了，执行sshd命令。
  5. 接下来查看手机的ip地址，输入ifconfig -a，可以看到你手机的ip地址形如192.168.1.2，记下来。
  6. 再查看Termux用户名，输入whoami，能看到形如u0_a82，就是你的用户名，记下来。

三、继续回到电脑上操作：

  1. 输入ssh u0_a82@192.168.1.2 -p 8022，然后一路回车，即可登录到你手机的Termux了。

杂谈

掌叔的第一部诺基亚手机NOKIA3510

大家一看到这个题目肯定一眼就看出这一定是掌叔遥想当年，怀旧式的一篇文章。的确，自小众网正式建立以来掌叔终于有一个可以静下心来写点东西的地方。除了分享掌叔开发方面的一些经验以及各种小众系统资讯的基础上，也希望能抒发一下深藏已久的感情。 写成文章一方面是满足掌叔多年以来的一种情怀，另一方面也希望能通过这种方式唤醒屏幕后的你同样心里埋藏的那份情怀。 说回正题，掌叔是在2003年的时候获得了人生中第一部诺基亚手机（掌叔第一步手机是摩托罗拉的v8088），那时候掌叔在上大二，手机是我一位亲戚送的。刚拿到它的时候，觉得样子很搞笑，尤其是外壳那蓝白条纹，这个造型曾被我同宿舍的一位同学戏称为裤衩子手机。你别说看着也确实有点像，再看它那不规则的按键造型，这和当时市面上其它手机相比显得很个性。刚入手的时候除了外观还有一点另我印象特别深刻，就是它的铃声，四和弦，那时候周围人用的手机铃声还都是只能发出滴滴答答的单音，而它四和弦的铃声太悦耳、太美妙了，尤其印象深的是有一首铃声听起来格外的空灵，仿佛身处广袤的宇宙深处。 手机内置的游戏也是一绝，印象最深的两个游戏一个是赛车，还有一个是空间大战。虽然是黑白单色屏幕，不过这俩游戏却做出了当年8位机的水准！有前景有背景，背景还是带卷轴的，想当初掌叔可是玩的不亦乐乎啊！一有空就来上一把。 至于其他功能我印象中是可以通过GPRS上网的，不过通过那个小屏幕上网的意义也不是很大，再加上那时候的网速慢、费用高，所以它的网络功能几乎是没用的。不过当时可以通过发送短信到服务商下载手机铃声，一条铃声一块钱，当时花了几块钱下了几首周杰伦歌做铃声，用了很长时间。好像一直到手机电池彻底充不进电了都没有再换过。 那时候的手机不像现在的智能机，数不清的软件，让人眼花缭乱，那时游戏机就是游戏机、相机就是相机、而手机真的就只是手机。在看看现在，在某种程度上手机已经是电脑+相机+游戏机了。虽然硬件强大、功能丰富，但却再也找不回当初那份单纯的快乐。 你还记得你离上次折腾手机已经过去多久了么？

使用WebWorks SDK开发Blackberry10应用

前面掌叔给大家简要介绍了通过安装Momentics IDE和Cordova以命令行的方式进行bb10应用的开发编译。（https://iniche.cn/63）因为全程涉及到的安装和命令步骤过于繁琐，估计已经有一部分同学望而却步了。那么有没有一种界面化、操作简单的方式呢？答案是有的。今天掌叔就带大家一起来看看黑莓官方的bb10应用开发工具：WebWorks SDK。 首先我们先到黑莓开发者官网下载WebWorks SDK安装包。 掌叔的系统是windows10，安装起来一切顺利。装好之后我们点击图标它会启动一个命令行窗口同时在你的浏览器里打开如图界面： 接下来我们一步一步根据提示进行操作。 1、创建项目。 2、配置项目，这里可以设置图标、启动图、开发者信息以及一些权限配置。 3、接下来就可以编译安装了，点击左侧菜单的建立。输入手机锁屏密码和你令牌的密码。之后点击“建立和安装”。 一切顺利的话就能看到你的手机已经启动这个应用了。

搭载KaiOS的手机JioPhone2手机已经发布

KaiOS系统的官方网址是：www.kaiostech.com/ JioPhone2是一款类似NOKIA E5那样的直板全键盘手机，它采用了一块240*320像素2.4寸的屏幕，配备了Qwerty全键盘，搭载了一枚1.1Ghz双核处理器，存储为512M内存和4G存储，还支持128GTF卡扩展，配备了前置30万像素后置200万像素镜头，电池容量为2000毫安时，支持4G网络。单从硬件配置来看，像极了当初的NOKIA手机。 可能很多朋友看到这些和今日主流的硬件相比已经十分孱弱的配置后，估计兴趣已然全无了，但是对掌叔来说这么一款低配手机却有着极大的吸引力。首先就是它搭载的KaiOS这个小众系统，大家都知道掌叔是一名不折不扣的小众系统迷，再者掌叔对全键盘更是天生没有抵抗力。其实它和一般意义上的功能机还是有很大不同的，毕竟KaiOS的前身可是FireFoxOS，也是一款智能操作系统呢，可玩性还是很高的。可惜这款手机只在印度发售，如果能在中国上市，掌叔肯定是买买买~

使用Cordova开发黑莓BlackBerry10应用

1、下载安装BlackBerry Native SDK（官网下载） 打开刚装好的Momentics IDE，会提示要下载SDK，这时候电脑接上黑莓bb10手机，然后选择Connect a BlackBerry 10 device 下一步，输入手机密码，注意这一步一定要开启手机的开发者模式。 下一步选择API级别，默认就好，点击Install。 开始下载了。 然后我们进入安装目录bbndk双击里面的bbndk-env_10_3_1_995.bat文件它会帮我们处理好系统环境设置。 2、安装完之后下一步我们来创建bbidtoken.csk令牌 首先我们要先创建一个黑莓开发者账户，这里掠过开发账户申请流程，我们假设看到这里的同学已经创建好了自己的黑莓开发者账号。 访问https://developer.blackberry.com/codesigning/用你的账号登录 进入这个界面开始创建令牌，这里需要输入一个6至8位的密码。为了好记可以输入你黑莓开发者账号的密码。 点击“获取令牌”之后浏览器就会自动下载一个bbidtoken.csk文件，根据页面提示我们将这个文件复制到％HOMEPATH％\ AppData \ Local \ Research In Motion这里。 然后打开一个命令行窗口输入：C:\bbndk\host_10_3_1_12\win32\x86\usr\bin\blackberry-keytool -genkeypair -storepass 123456 -author iniche，执行成功后会在刚才令牌那个文件夹里生成一个author.p12文件。123456替换成你令牌的密码，iniche替换成你要的令牌作者名称，最好是英文名称。 接下来我们来安装cordova，首先要先安装好nodejs，我们可以到nodejs的官网(https://nodejs.org/en/)去下载，nodejs有安装包，安装很简单，这里就不再赘述了。当然注意一下版本，掌叔用的是8.2.1版本的。 装好nodejs之后我们打开一个CMD命令行窗口，在里面输入npm install -g cordova@6.1，电脑开始自动安装cordova需要文件，过一会就会提示安装完毕。 一切都装好之后下面就让我们来创建一个项目吧。 一次在命令行窗口输入如下命令： $ cordova create hello com.example.hello $ cd hello $ cordova platform add blackberry10 $ cordova build 如果一切顺利的话就能看到bar文件打包成功的提示，如下图。 安装包已经打包成功了，接下来我们把应用部署到我们的手机里吧。 首先看一下我们手机的PIN码是多少，方法是系统设置-关于-类别选硬件。然后把PIN码记下来。然后把手机用USB线和电脑连接，在手机上进入安全和隐私-开发模式，打开开发模式。这里有一个IP地址我们也记下来。 下面我们在手机上安装调试令牌，并且安装应用。输入cd platforms\blackberry10\cordova回车切换到这个文件夹。然后输入run.bat –device –keystorepass mypassword，mypassword要替换成你令牌的密码。执行成功后就会在手机里安装调试令牌，并且自动帮你把刚才的hello项目安装到你的手机里了。一切正常话，应用汇自动启动，如下图这样：
